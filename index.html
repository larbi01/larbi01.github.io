Secure File Transfer & Integrity Verification (SFTIV)

Python WSL AES-256-GCM SHA-256 Ed25519

Developed a Python-based cybersecurity tool that ensures secure, verifiable file transmission by combining AES encryption, SHA-256 integrity validation, and Ed25519 digital signatures. The project demonstrates practical implementation of the Confidentiality, Integrity, and Authenticity (CIA) principles in data exchange.

Key Features:

  • AES-256-GCM file encryption for strong confidentiality
  • SHA-256 manifest generation and verification for data integrity
  • Ed25519 digital signatures to confirm file authenticity
  • Lightweight TCP client/server model for local or network transfers
